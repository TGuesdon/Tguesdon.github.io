Twitch interactive quizz

A quizz that can be streamed by twitch streamer, with viewers able to answer directly on twitch chat.

This was developped using NodeJS, Angular, TMI.js and Firebase.

TG's E-commerce photo cleaner

TG's E-commerce photo cleaner is an experimental project on which I tested if I could clean ecommerce product photos. In order to do that I used my knowledge from computer vision studies and the openCV library.

Idealtek
Developpement and maintainance of Idealtek e-commerce website.
SpaceConquest
Space Conquest is a Java project created during our final year of an IT bachelor's degree in University of Bordeaux (FR). It's a strategy game where you are the Ambassador of the red planet and will have to conquest all planets around. It contains an Implementation of the A* algorithm.
Menulib
Menulib is a WIP project which allows to order with a qrCode at the restaurant. It is different than other similar applications because the qrCode is displayed on a screen and refreshed everytime someone use it. This allows far better security than qrCode on paper.
Mashup Helper
Mashup Helper is a small angular / firebase project I did to help a friend of mine do his songs mashups.
Blindtestrap
Blindtestrap is a website on which you can play a difficult blindtest on french rap songs, with their videos getting less and less blurred with the countdown arriving to the end.
BuildIdeas
BuildIdeas is a Minecraft plugin allowing player to get randomly generated ideas using the command /bidea !
